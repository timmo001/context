The `context` CLI is the direct interface to the same producers exposed through MCP.

```bash
context git
context stack
context mcp
```

Use `context help` or `context <command> --help` for terminal help. The [command reference](/cli/commands/) is generated from the same CLI registry.

Argument parsing is strict. Unknown or repeated options, missing values, invalid fixed choices, and extra positional arguments return a usage error. Use `--` before a positional path that begins with a hyphen.

Git and GitHub subprocesses default to a 30-second timeout and a 16 MiB aggregate output limit. Set `CONTEXT_COMMAND_TIMEOUT_MS` or `CONTEXT_COMMAND_MAX_OUTPUT_BYTES` to non-negative integer values when a repository needs different limits.

Shell completions are generated by [`context completions`](/cli/completions/).
