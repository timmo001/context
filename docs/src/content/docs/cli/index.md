The `context` CLI is the direct interface to the same producers exposed through MCP.

```bash
context git
context stack
context opencode-debug
context mcp
```

Use `context help` or `context <command> --help` for terminal help. The [command reference](/cli/commands/) is generated from the same CLI registry.
